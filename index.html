APA ITU PWA??

Progressive Web App (PWA) adalah aplikasi web yang dikembangkan dengan teknologi web modern namun menawarkan pengalaman seperti aplikasi native. PWA bisa diakses melalui browser tetapi memiliki kemampuan layaknya aplikasi yang diinstal di perangkat.


Fitur PWA

  • Responsif: Berfungsi dengan baik di berbagai ukuran layar dan perangkat
  • Offline: Menggunakan Service Worker untuk menyimpan data sehingga bisa diakses tanpa koneksi internet
  • Installable: Bisa diinstal langsung dari browser tanpa perlu melalui app store
